SEO plugin for indexing WordPress content and monitoring search engine performance.

== Description ==

[CrawlWP](https://crawlwp.com/) boosts and monitors your WordPress site’s visibility on Google, Bing, Yandex, Naver, Yep, and Seznam.cz with our SEO insights and automatic indexing features.
